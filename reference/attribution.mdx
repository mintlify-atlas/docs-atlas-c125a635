## Attribution Requirements

When using or adapting this application, you must:

1. **Credit the original RAADS-R authors:**
   - Include the academic citation (see above)
   - Link to the original paper when possible

2. **Credit this implementation:**
   - Link to the [GitHub repository](https://github.com/spike1478/raads-r-self-host)
   - Indicate if you made changes to the code

3. **Include license information:**
